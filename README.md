# Dart 예외처리 실습 프로젝트

이 저장소는 Dart 언어를 활용해 작성한 세 가지 실습 문제를 포함하고 있으며,  
각 문제는 **특수 상황에 대한 예외 처리(2가지 이상)**를 구현하는 것을 목표로 한다.

---

## 1. 점수 입력 후 등급 출력 프로그램

### ✔ 설명  
사용자로부터 점수를 입력받아 A/B/C 등급을 출력하는 프로그램.  
여러 특수 상황에 대비해 입력 검증 및 예외 처리를 구현하였다.

### ✔ 예외 처리 내용
- 입력이 `null` 또는 빈 문자열일 경우 경고 후 종료  
- 숫자로 변환 불가능한 값을 입력한 경우 처리  
- 점수가 0~100 범위를 벗어날 경우 오류 처리


## 2. 장바구나 가격 계산 프로그램

### ✔ 설명  
사용자로부터 점수를 입력받아 A/B/C 등급을 출력하는 프로그램.  
여러 특수 상황에 대비해 입력 검증 및 예외 처리를 구현하였다.

### ✔ 예외 처리 내용
- 입력이 `null` 또는 빈 문자열일 경우 경고 후 종료
- 할인 조건 예외 처리: 금액이 0이거나 음수일 리 없지만 안정성 확보


## 2. 장바구나 가격 계산 프로그램

### ✔ 설명  
랜덤으로 6개의 로또 번호를 생성하고, 미리 지정된 당첨 번호와 비교해 등수를 판정한다.

### ✔ 예외 처리 내용
- 발급 번호가 유효한지 검사
